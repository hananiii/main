# eleave-v3

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ ionic serve
```

## Documentation

```bash
# Build documentation
$ npm run doc:build

# Build and serve documentation on http://localhost:8080
$ npm run doc:serve
```
